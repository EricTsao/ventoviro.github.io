Windwalker

A Modern PHP Framework

Features

Lightweight and Elegant

Windwalker is very fast, we use many design patterns like Lazy loading or cache to make sure we have high performance.

Easy But Powerful

We know that learning a new framework is very hard and cost long time, so Windwalker's interface is very simple and semantic that will not make developer pain.

Convention over Config

Windwalker try to reduce configuration work, it use minimal config to extend whole system. But you can still override every part of Windwalker to customize your application.

Fully Decoupled

Windwalker packages are very cohesion, we are trying to make every package dose not depend on too many other packages. You can easy override Windwalker's behavior but don't worry about dependencies.

Extendable

The Windwalker package system (like Symfony bundles) help us create complex application for our enterprise. Also you can create your package as an extension to let other developers install it.

HHVM Ready

After hard working, Windwalker has supported HHVM server and pass all tests. You can run Windwalker in HHVM server and start your programming.

Quick Start

$ php composer.phar create-project windwalker/starter your_project 2.*
